# fiber-webdav

fiber-webdav is a fork of [golang.org/x/net/webdav](https://pkg.go.dev/golang.org/x/net/webdav). This project adapts the
original WebDAV implementation to the [Fiber](https://gofiber.io/) framework and includes code cleanup.

## Installation

Install via `go get`:

```bash
go get github.com/Tryanks/fiber-webdav
```
